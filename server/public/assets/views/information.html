The Conscious Shopper App is an offshoot of ethical lifestyle and resource blog Life+Style+Justice. Created out of a desire to connect consumers with local shops carrying ethically sourced goods, our goal is to crowdsource the world’s largest interactive database of conscious shopping venues. You can learn more about the project here.

If you’d like to get in touch with the Conscious Shopper App team to ask a question or submit a favorite ethical store to add to the map, use the form below. (no online-only stores, please! Conscious Shopper is a resource for brick-and-mortar shops)

iPad and iPhone with Safari

  1. From the Conscious Shopper app select the bookmark icon
  2. Select ‘add to home screen’ when bookmark options appear
  3. Change the shortcut name to ‘Conscious Shopper’
  4. Conscious shopper shortcut will appear on home screen

Android or iPhone with Chrome

  1. From the Conscious Shopper app select the menu button
  2. (three vertical dots next to the URL)
  3. From the menu select ‘add to Home screen’
  4. Conscious shopper shortcut will appear on home screen

Android or iPhone with Firefox

  1. From the Conscious Shopper app select the title bar
  2. From the title menu select ‘Bookmarks’
  3. Long Tap the bookmark (hold it for a little longer) and select “Add to Home Screen”
  4. Conscious shopper shortcut will appear on home screen
